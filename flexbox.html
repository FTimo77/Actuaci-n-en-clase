Flexbox

Flexbox, o Flexible Box Layout, es un modelo de diseño CSS que proporciona una forma eficiente de distribuir elementos dentro de un contenedor y alinearlos de manera dinámica.
Antes de Flexbox, los desarrolladores tenían opciones limitadas, como el uso de float, que no eran compatibles entre navegadores y difíciles de escalar para adaptarse a diferentes resoluciones de pantalla. Flexbox supera estas limitaciones al ofrecer un conjunto de propiedades y valores que permiten centrar bloques vertical y horizontalmente, asegurando además que todas las columnas de un contenedor tengan la misma altura, independientemente de la cantidad de contenido que contengan.

Estructura

La estructura de Flexbox se compone de contenedores padre e hijos (Contenedor-Flex y Elementos-Flex respectivamente).

  • Contenedor-Flex: Es el elemento padre que contiene los elementos hijos. Se define con la propiedad CSS display: flex;.
  • Elementos-Flex: Son los elementos hijos dentro del contenedor. Se adaptan automáticamente al espacio disponible en el contenedor.

Flex-direction

Es la encargada de definir el eje principal y secundario de nuestros elementos hijos. Estos ejes, como ya se mencionó, pueden ser verticales, formando columnas, y horizontales, formando filas. 


.contenedor-padre { 
  display: flex;
  flex-direction: row | row-reverse | column | column-reverse;
}

Flex-wrap

Por defecto, flex trata de disponer los elementos en una misma línea. Si no es el caso que necesitas, con flex-wrap puedes ordenar los elementos en más de una fila o columna. 


.contenedor-padre {
  display: flex;
  flex-wrap: nowrap | wrap | wrap-reverse;
}

Justify-content

Cuando nombramos el eje principal de un elemento padre, es decir, el eje horizontal, se destaca esta propiedad para dar direccionamiento a los elementos hijos. 


.contenedor-padre {
  display: flex;
  justify-content: stretch | flex-start | flex-end | center | space-between | space-around | space-evenly;
}

Align-items

Esta propiedad alinea los elementos hijos a lo largo del eje secundario (por defecto, el eje vertical). Es útil para controlar la alineación vertical de los elementos dentro del contenedor flex. 


.contenedor-padre {
  display: flex;
  align-items: stretch | flex-start | flex-end | center | baseline;
}

Align-content

Esta propiedad distribuye el espacio entre las líneas del contenedor flex cuando hay varias líneas (por ejemplo, cuando se usa flex-wrap: wrap). No afecta si solo hay una línea de elementos. 


.contenedor-padre {
  display: flex;
  flex-wrap: wrap;
  align-content: stretch | flex-start | flex-end | center | space-between | space-around;
}

Propiedades de los elementos hijos (Flex Items)

Los elementos hijos de un contenedor flex también tienen propiedades específicas para controlar su tamaño y alineación:

  • order: Cambia el orden visual de los elementos.
  • flex-grow: Define cuánto crecerá un elemento en relación a los demás.
  • flex-shrink: Define cuánto puede reducirse un elemento si es necesario.
  • flex-basis: Define el tamaño base del elemento antes de distribuir el espacio restante.
  • align-self: Permite alinear individualmente un elemento diferente al resto. 

.elemento-hijo {
  order: 2;
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 200px;
  align-self: center;
}
